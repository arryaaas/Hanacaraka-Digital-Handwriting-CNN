# Hanacaraka-Digital-Handwriting-CNN

Pengenalan Tulisan Tangan Digital Hanacaraka Menggunakan Convolutional Neural Network

## Dataset

```
└─── hanacaraka
    |─── ha ( 510 gambar )
    |─── na ( 510 gambar )
    |─── ...
    └─── nga ( 510 gambar )
```
